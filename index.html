워크멘토 소개

중소기업의 두 가지 고민, "사람"과 "돈"을 잇는 멘토가 되겠습니다.

대한민국 중소기업 대표는 광범위한 정책 발표 및 채용에 관련된 플랫폼에서 꼭 필요한 지원정책과 인재를 찾기위해 언제나 고공분투를 하지만 쉽지 않은 방법으로 인해 결과와 인연을 포기를 합니다. 워크멘토는 기업에 대한 모든것을 개별적으로 지원합니다. 또한 인재 채용 정보, 기업 간 네트워킹, 정부 지원사업에 관한 최신 데이터와 정보를 모든 기업과 대표님들을 위해 모아 더 빠른 성장을 설계할 수 있도록 돕는 통합 플랫폼입니다.

관리자 대시보드

기업·채용공고 등록/수정/삭제, 회원 등급 관리, API 연동 설정을 할 수 있습니다.

기업마당 실시간 API 연동 방법
1) 기업마당(bizinfo.go.kr) → 활용정보 → 정책정보 개방에서 API 인증키를 무료로 신청합니다.
2) 발급받은 인증키를 아래에 입력하고 저장하면 지원사업 페이지가 실시간 공고로 전환됩니다.
3) 브라우저 보안(CORS) 때문에 직접 호출이 막히면, 함께 드린 bizinfo-proxy-worker.js를 Cloudflare Workers(무료)에 배포하고 그 주소를 프록시 URL에 넣으세요. 자세한 방법은 README 참고.

지금 버전은 데이터가 이 브라우저 안(localStorage)에 저장됩니다. 다른 컴퓨터나 방문자에게는 보이지 않으니, 등록을 마친 뒤 내보내기로 JSON 파일을 받아 저장소에 함께 올리거나, 다음 단계에서 Supabase 같은 무료 DB로 옮기면 모두에게 공유됩니다.

⚠ 이 로그인은 화면 잠금 수준의 보호입니다. 실서비스의 진짜 보안(서버 인증)은 다음 단계인 Supabase 연동에서 적용됩니다.

⚠ 결제 데모 화면입니다. 실서비스에서는 사업자등록 후 토스페이먼츠 등 PG사 심사를 거쳐 이 자리에 실제 결제창이 연동됩니다.
